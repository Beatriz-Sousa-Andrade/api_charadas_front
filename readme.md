# Gerador de Charadas • Mistério

**Descrição:**
O projeto consiste em uma aplicação interativa de charadas ("enigmas") que utiliza uma API externa para fornecer conteúdo dinâmico. O objetivo é oferecer um desafio cerebral divertido com uma interface moderna e responsiva, onde o usuário pode testar seus conhecimentos, acumular pontos e compartilhar os resultados.

## Funcionalidades
- **Busca de Charadas:** Consome uma API assíncrona para buscar novas perguntas e respostas a cada rodada.
- **Card Interativo 3D:** Interface com efeito de rotação que esconde a resposta até o clique do usuário.
- **Sistema de Pontuação:** Acompanhamento de progresso em tempo real (Ganha 10 pontos por acerto e perde 5 por erro).
- **Compartilhamento Automático:** Copia a charada e a resposta para a área de transferência para facilitar o envio em redes sociais.
- **Interface Responsiva:** Desenvolvida com sistema de grid para se adaptar perfeitamente a qualquer tamanho de tela ou dispositivo.

## Tecnologias Utilizadas
- **Linguagem/Frameworks:**
 - **HTML5**: Estruturação semântica da aplicação.
 - **Tailwind CSS**: Estilização baseada em classes utilitárias para um design moderno e responsivo.
 - **JavaScript (ES6+)**: Lógica de manipulação do DOM, controle de pontuação e requisições à API.
 - **Google Fonts**: Utilização da fonte 'Space Grotesk' para uma identidade visual limpa.
